# MNIST Classification: MLP vs CNN

## Описание
Репозиторий "**mnist-classification**" содержит реализацию и сравнение двух нейросетевых архитектур для задачи классификации рукописных цифр MNIST.
- Multilayer Perceptron (MLP)
- Convolutional Neural Network (CNN)


## Стэк технологий
- Python
- PyTorch
- PyCharm, Jupyter Notebook


## Результаты
1. Для обеих архитектур выбраны:
   - Loss: CrossEntropyLoss()
   - Optimizer: Adam()
2. Итоги обучения (5 эпох):
   - MLP: Accuracy - 98.0%, Avg loss - 0.079482
   - CNN: Accuracy - 99.0%, Avg loss - 0.032220

   Важно: seed не был установлен, поэтому при повторении эксперимента могут получиться иные результаты.

3. Проведено тестирование моделей на:
   - стандартном датасете MNIST
   - собственных изображениях рукописных цифр

   Несмотря на различия между моими изображениями и данными MNIST, обе модели смогли их классифицировать. Однако CNN показала значительно более хорошие результаты по сравнению с MLP.
   Возможные улучшения включают доработку preprocessing (центрирование, бинаризация, нормализация) и изменение архитектуры MLP (увеличение глубины, регуляризация), однако для задач классификации изображений CNN в целом является более подходящей моделью
